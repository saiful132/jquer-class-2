how can we see the attribute of 'a' tag?

ans:

JS CODE = document.querySelector("a").getAttribute("href")

JQUERY CODE = $("a").attr("href")

function effect1() { var jsEffect = document.querySelector(".a").getAttribute("href"); document.querySelector(".p1").innerHTML = jsEffect; }

Effect will show here

function effect2() { var jsEffect2 = $(".a").attr("href"); $(".p2").text(jsEffect2); }

Effect will show here

[They are same]
